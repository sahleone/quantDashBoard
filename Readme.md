📘 Project Overview

This application was developed as part of a capstone project for the Springboard Software Engineering Program. It brings together concepts from software engineering and quantitative finance to create a full-stack platform where users can track and analyze investment portfolios.

🎯 Purpose

The goal of this project is to help users understand portfolio performance and risk using real financial data and proven models. It also serves as a demonstration of practical skills in building secure, scalable web applications with data-driven insights.

🧱 What It Covers
• Full-Stack Development using the MERN stack (MongoDB, Express.js, React.js, Node.js)
• Authentication with JWT to ensure secure user access
• REST API Design for handling portfolios, user sessions, and analytics
• Real-Time Data Integration from APIs like Alpha Vantage, SnapTrade
• Financial Metrics such as CAGR, Sharpe Ratio, Beta, and Max Drawdown
• Interactive Visualization of performance, volatility, correlation, and factor exposures

🧠 Key Concepts Applied
• Time Series Analysis
• Risk-Adjusted Return Metrics
• Data Visualization and User Experience Design
• Secure Backend Architecture and Session Management

📈 What Users Can Do
• Log in securely and manage their investment portfolios
• View dynamic charts of portfolio growth and historical performance
• Analyze exposure to market factors and risk metrics
